Eliot - Ransomeware Tool

Project information

  • Category Cyber Security and Forensics
  • Visit Github Repository

Recent cybersecurity project that's been a significant part of my ongoing journey to deepen my knowledge in the field. I've developed an encryption tool in Python, and I've named it 'Eliot.' This tool utilizes the Fernet cryptography library to encrypt files in the current directory, effectively rendering them inaccessible. The only way to restore these files is by using the corresponding decrypter and the unique key generated during encryption.
